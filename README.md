## 🚀 Cài Đặt & Chạy

### 📋 Yêu Cầu Hệ Thống

```bash
✓ Node.js v14+         (https://nodejs.org)
✓ NPM v6+              (đi kèm Node.js)
✓ Python 3.8+          (cho FFmpeg)
✓ FFmpeg               (xử lý media)
✓ Tài khoản Zalo       (để đăng nhập)
✓ 500MB RAM tối thiểu   (chạy bot)
✓ Kết nối Internet     (24/7)
```

### 💻 Cài Đặt Chi Tiết

#### **Bước 1: Clone Repository**

```bash
# Sử dụng Git
git clone https://github.com/NQD136/Nqduan-share-zalobot.git
cd Nqduan-share-zalobot

# Hoặc tải ZIP
# https://github.com/NQD136/Nqduan-share-zalobot/archive/main.zip
```

#### **Bước 2: Cài Đặt Dependencies**

```bash
# Cài đặt npm packages
npm install

# Hoặc sử dụng yarn
yarn install
```

> ⚠️ **Lưu ý:** Quá trình này có thể mất 5-10 phút tùy tốc độ internet

#### **Bước 3: Cấu Hình Bot**

```bash
# Sao chép file cấu hình mẫu
cp assets/config.json.example assets/config.json

# Mở file cấu hình
# Windows: notepad assets/config.json
# Mac/Linux: nano assets/config.json
```

**Chỉnh sửa `assets/config.json`:**

```json
{
  "cookie": "YOUR_ZALO_COOKIE_HERE",
  "imei": "YOUR_DEVICE_IMEI",
  "userAgent": "Mozilla/5.0...",
  "admins": ["YOUR_UID"],
  "prefix": "/",
  "debug": false
}
```

> 🔍 **Cách lấy cookie Zalo:**
>
> 1. Mở Zalo Web (https://chat.zalo.me)
> 2. Mở DevTools (F12)
> 3. Tab Network → Xem cookie `imei`, `sessionId`

#### **Bước 4: Chạy Bot**

```bash
# Chạy bình thường
npm start

# Hoặc chạy với PM2 (khuyến nghị)
npm run gopm2

# Chạy bot phụ
npm run bot
```

> ✅ **Nếu thành công, bạn sẽ thấy:**
>
> ```
> ✓ Bot đã kết nối
> ✓ Đang nghe tin nhắn...
> ```

## ⚠️ Lưu Ý & Best Practices

### 🚨 Cảnh Báo Quan Trọng

> ⛔ **KHÔNG NÊN:**
>
> - ❌ Spam tin nhắn quá nhanh (bot bị khóa)
> - ❌ Chia sẻ cookie/token của bot công khai
> - ❌ Sử dụng bot để spam/scam/lừa đảo
> - ❌ Vi phạm Điều khoản dịch vụ Zalo
> - ❌ Hack/crack các tính năng khác

> ✅ **NÊN LÀM:**
>
> - ✓ Backup dữ liệu thường xuyên
> - ✓ Cập nhật bot thường xuyên
> - ✓ Đọc log để phát hiện lỗi sớm
> - ✓ Tuân theo pháp luật & đạo đức
> - ✓ Sử dụng proxy an toàn

### 🔒 Bảo Mật

```bash
# Bảo vệ file config
chmod 600 assets/config.json

# Sử dụng biến môi trường
export ZALO_COOKIE="your_cookie"

# Mã hóa dữ liệu nhạy cảm
npm install crypto-js

## 🐛 Xử Lý Lỗi & Troubleshooting

### ❌ Lỗi Phổ Biến

| Lỗi                  | 原因               | Giải Pháp                 |
| -------------------- | ------------------ | ------------------------- |
| `ENOENT`             | File không tồn tại | Kiểm tra path, tạo folder |
| `Cannot find module` | NPM package thiếu  | Chạy `npm install` lại    |
| `Socket hang up`     | Mất kết nối        | Kiểm tra proxy, VPN       |
| `Permission denied`  | Quyền hạn          | `chmod +x` hoặc chạy sudo |
| `Out of memory`      | RAM không đủ       | Tăng RAM hoặc tối ưu code |

### 🔧 Debug Commands

```bash
# Kiểm tra phiên bản Node.js
node --version

# Kiểm tra npm
npm -v

# Kiểm tra cấu hình
npm run config

# Test API Zalo
npm run test-api

# Xóa cache NPM
npm cache clean --force
```
